Launch the tool [here](https://kjy5.github.io/artifact-meta-manager/).

## Development

1. Get the latest version of [Node.js](https://nodejs.org/en/) (LTS or Current).
2. Clone the repository.
3. `cd artifact-meta-manager`
4. `npm install`
5. `npm run dev`
